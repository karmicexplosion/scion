scion by nominolo

An IDE library for Haskell based on the GHC API.

The goal of Scion is to provide Haskell IDE functionality as a Haskell library or service. Scion aims to be frontend agnostic, so that we can have IDE like features for many different editors. Scion comes with an Emacs frontend, but it is intended that Scion be used as the backend for Vim plugins, Eclipse plugins, or many other scriptable editors as well.

Install

To install a release version of Scion, use cabal-install: 

$ cabal update && cabal install scion

Download

To get the latest release version use cabal-install: 

$ cabal update && cabal install scion
If you want to look at the sources for that release, use: 
$ cabal unpack scion

To get the latest development version clone the project with Git by running: 

$ git clone git://github.com/nominolo/scion
